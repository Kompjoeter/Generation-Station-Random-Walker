Generation Station: Random Walker

A small generative program which creates paths with the Random Walker technique coupled with the P5 Library and a canvas to draw the generated paths onto (in three unique ways).

I made this program to get familiar with the P5 Library and coupled my Template Canvas-UI to it, which can be found as a standalone project on my GitHub.
